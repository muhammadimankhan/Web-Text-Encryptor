# Web-Text-Encryptor
# Secure Web-Based Text Encryption Tool

## Overview
A lightweight, single-page web application that allows users to encrypt, decrypt, and hash text directly in their browser. Built as part of a university project to demonstrate practical applied cryptography.

## Features
* **Multiple Algorithms:** Supports AES, Caesar Cipher, and Base64 encoding.
* **Hashing (Bonus Feature):** Includes one-way SHA-256 hashing.
* **Two-Way Processing:** Fully supports both encryption and decryption for symmetric algorithms.
* **Dynamic UI:** Interface adapts based on the selected algorithm's requirements (e.g., hiding the secret key field when not needed).
* **Clipboard Utility:** One-click copying of output text.
* **Validation:** Prevents empty inputs and ensures proper algorithm selection.

## Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Cryptographic Library:** [CryptoJS](https://cryptojs.gitbook.io/docs/)

## How to Run
Because this is a completely client-side application, no server setup or Node.js installation is required. 
1. Clone this repository to your local machine.
2. Open the `index.html` file directly in any modern web browser (Chrome, Firefox, Edge).
